内存类型
1.栈内存，申请和释放都是编译器隐式完成的
void func() {
    int x; # 进入func函数时，栈向上开辟空间，离开函数时，栈向下释放空间
}
2.堆内存，申请和释放都是显式完成的，由malloc()和free()函数完成
void func() {
    int *x = malloc(sizeof(int)); # 栈和堆的分配同时在这一行发生, 因为编译器要为(int *x)整形指针分配空间
    free(x); # 分配区域的大小不会被用户传入，由内存分配库本身记录追踪
}

常见错误
1. 忘记分配内存
2. 没有分配足够内存
3. 忘记初始化分配的内存
4. 忘记释放内存
5. 在用完之前释放内存
6. 重复释放内存
7. 错误调用free()

sizeof()是一个运算符，编译时就会计算出分配的内存大小
malloc()和free()是库函数不是系统调用
brk()和sbrk()是系统调用，用于调整堆的大小
mmap()是系统调用，创建一个匿名内存区域，与swap空间关联
